# Initial_setup
Basic python package setup for new conda environment

When creating new conda environment, simply run below command

## Installation using requirement
```bash
pip install -r requirements.txt
```

## Installation
```bash
ptyhon setup.py install
```

## Included Package
* numpy>=1.15.4',
* scipy>=1.1.0',
* os
* glob
* datatime
* matplotlib
* scikit-learn
* pandas

## Further requirement
* tensorflow (>0.12)
